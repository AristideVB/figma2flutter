
# figma2flutter

Convert [Tokens Studio for Figma]() json exports to Flutter code.

## Token support

- [ ] sizing
- [x] spacing
  - [x] 1, 2, 3 or 4 values
  - [x] accept int/double values, px of rem suffix
- [ ] color
  - [x] Solid
    - [x] hex
    - [x] rgb 
    - [x] rgba
    - [x] hsla
  - [ ] Gradients
- [ ] borderRadius
- [ ] borderWidth
- [ ] boxShadow
- [ ] opacity
- [ ] fontFamilies
- [ ] fontWeights
- [ ] fontSizes
- [ ] lineHeight
- [ ] letterSpacing
- [ ] paragraphSpacing
- [ ] textCase
- [ ] textDecoration
- [ ] typography
- [ ] asset
- [ ] composition (pro)
- [ ] dimension
- [ ] border
